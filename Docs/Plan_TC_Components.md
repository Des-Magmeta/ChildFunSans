# 總體計劃
 - 由於時間和精力關係，目前計劃僅包含 IICore 範圍内 9810 個漢字和 JF7000 基本包收錄漢字，并未考慮與「遊趣體」簡體版所收字彙平齊。（鑒於 IICore 覆蓋範圍有限，後續可能會視情况增加一系列字。） 
 - CL、MN 横排和直排均采用正中標點，蝌蚪引號采用西文式。 
 - 部分部件的字形視為設計特性，不做改動，如「穴」「令」「瓦」等字；
 - 同一部件的不同設計形態不構成衝突，如「穴」與「夋」「甚」内部的「八／儿」等。
